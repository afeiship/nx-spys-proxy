# nx-spys-proxy
> Spys proxy.

## installation
```rb
# from gem
gem 'nx-spys-proxy'
# from git
gem 'nx-spys-proxy', git: 'git@github.com:afeiship/nx-spys-proxy.git'
```

## usage
```rb
Nx::SpysProxy::hello

# hello world
```

## build/publish
```shell
# build
gem build nx-spys-proxy.gemspec

# publish
gem push nx-spys-proxy-0.1.0.gem
```
